キーバインド

Cmd-E または Tab Expand abbreviation(省略コードの展開)
Cmd-D Balance Tag (matches opening and closing tag pair)(要素の外側を選択)
Shift-Cmd-D Balance Tag Inward(要素の内側を選択)
Shift-Cmd-A Wrap With Abbreviation(省略コードで包括)
Ctrl-Alt-Right Next Edit Point(次の編集箇所への移動)
Ctrl-Alt-Left Previous Edit Point(前の編集箇所への移動)
Cmd-L Select line(行を選択)
Cmd-Shift-M Merge Lines(複数の要素を1行にする)
Cmd-/ Toggle Comment(コメントアウトの実行・解除)
Cmd-J Split/Join Tag(タグの分割・結合)
Cmd-K Remove Tag(タグの削除)
Shift-Cmd-Y Evaluate Math Expression(数式を評価)
Ctrl-Up Increment Number by 1(1増やす)
Ctrl-Down Decrement Number by 1(1減らす)
Ctrl-Alt-Up Increment Number by 0.1(0.1増やす)
Ctrl-Alt-Down Decrement Number by 0.1(0.1減らす)
Shift-Ctrl-Up Increment Number by 10(10増やす)
Shift-Ctrl-Down Decrement Number by 10(10減らす)
Shift-Cmd-. Select Next Item(次のアイテムを選択)
Shift-Cmd-, Select Previous Item(前のアイテムを選択)
Cmd-B Reflect CSS Value(CSSの値の反映)
